Array length : 배열의 크기 Array toString() : 배열의 요소를 콤마(,) 구분자로 문자열 반환 Array at() Array join() Array push() : 배열의 새로운 요소를 마지막에 추가하고 배열 크기를 반환 Array pop() : 배열의 마지막 요소를 제거하고 반환 Array shift() : 첫번째 요소를 제거하고 반환... 모두 왼쪽으로 이동...
